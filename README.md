# 🚀 Energy-Efficient Fraud Detection using Adaptive Sampling

## 📌 Project Overview

This project focuses on detecting fraudulent transactions using machine learning models while maintaining energy efficiency. It uses adaptive sampling techniques and provides an interactive Streamlit UI.

## 🚀 Features

* Fraud detection using ML models
* Adaptive sampling approach
* Interactive Streamlit UI
* Carbon footprint tracking using CodeCarbon

## 🧠 Models Used

* Logistic Regression
* Decision Tree
* Random Forest
* Support Vector Machine (SVM)
* XGBoost

## 🖥️ How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
